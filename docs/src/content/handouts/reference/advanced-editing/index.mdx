Markdown supports many extensions that adds other visual elements to text. In this page we describe the ones supported by _Interactive Handouts_.

## Box elements

Box elements can be use to emphasize text and add interactivity to a handout. All follow the same syntax:

```html
<ah-admonition title="Title goes here">
  Content goes here
</ah-admonition>
```

See examples of static box elements below.

<ah-tabbed>
  <label slot="label">Result</label>
  <div slot="content">
    <ah-admonition title="Note" type="note">
      You can create different types of box:
      
      - note, example, warning, info, tip, danger

      You only need to change `type` property.
    </ah-admonition>

    <ah-admonition title="Set the title" type="example">
      You can set the box title using the `title` property.

      <ah-admonition title="Nested boxes" type="warning">
        A box inside a box is also possible.
      </ah-admonition>
    </ah-admonition>

    <ah-admonition type="danger">
      This is a box without title.
    </ah-admonition>

    <ah-admonition title="Collapsible boxes" type="info" collapsible collapsed>
      Collapsible boxes are also possible.
    </ah-admonition>
  </div>

## Code and Syntax Highlighting

### Inline code

<ah-tabbed>
  <label slot="label">Result</label>
  <div slot="content">
  Text surrounded by backticks (the character) is shown in monospaced font, `like this`.
  </div>

  <label slot="label">Markdown</label>
  <div slot="content">
  ```md
  Text surrounded by backticks (the character) is shown in monospaced font, `like this`.
  ```
  </div>
</ah-tabbed>

### Code blocks

Code blocks are started and ended by a line with three backticks. See examples below.

<ah-tabbed>
  <label slot="label">Result</label>
  <div slot="content">
  No highlight:

  ```
  #include <stdio.h>
  int main(void) {
    printf("Hello world!\n");
    return 0;
  }
  ```

  Code highlight:

  ```c
  #include <stdio.h>
  int main(void) {
    printf("Hello world!\n");
    return 0;
  }
  ```
  </div>

  <label slot="label">Markdown</label>
  <div slot="content">
  ````md
  No highlight:

  ```
  #include <stdio.h>
  int main(void) {
    printf("Hello world!\n");
    return 0;
  }
  ```

  Code highlight:

  ```c
  #include <stdio.h>
  int main(void) {
    printf("Hello world!\n");
    return 0;
  }
  ```
  ````
  </div>
</ah-tabbed>

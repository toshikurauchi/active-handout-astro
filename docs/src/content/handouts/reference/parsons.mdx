You can also create Parsons problems. For more details and examples, please refer to the following resources:

- https://pamelafox.github.io/faded-parsons-static/
- https://dl.acm.org/doi/10.1145/3411764.3445228

## Getting Started

To utilize Parsons problems, start by providing an explanation to the student, followed by the lines of the exercise. The plugin will automatically shuffle the lines for you.

<Parsons slug="parsons-code" withIndentation>

Write a function that prints the following and returns the number `#!python 4`:

```
abc
2
3
4
```

<ParsonsLine>`#!python def teste():`</ParsonsLine>
<ParsonsLine indent={1}>`#!python print('abc')`</ParsonsLine>
<ParsonsLine indent={1}>`#!python for i in range(3):`</ParsonsLine>
<ParsonsLine indent={2}>`#!python print(i+2)`</ParsonsLine>
<ParsonsLine indent={1}>`#!python return 4`</ParsonsLine>

</Parsons>

The indentation is optional:

<Parsons slug="parsons-without-indent">

Print the lines in ascending order. The string `distractor` should not be printed.

<ParsonsLine>`#!python print("line 1")`</ParsonsLine>
<ParsonsLine>`#!python print("line 2")`</ParsonsLine>
<ParsonsLine>`#!python print("line 3")`</ParsonsLine>
<ParsonsLine>`#!python print("line 4")`</ParsonsLine>
<ParsonsLine distractor>`#!python print("distractor")`</ParsonsLine>

</Parsons>

If more than one answer is valid, you can also use Python unittests!

<Parsons slug="parsons-with-unittests" withIndentation pythonTestsSrc="/parsons/unit_test_example.py">

Define a function that returns the received number plus 2.

<ParsonsLine>`#!python def add_two(n):`</ParsonsLine>
<ParsonsLine>`#!python n += 2`</ParsonsLine>
<ParsonsLine>`#!python return n`</ParsonsLine>
<ParsonsLine>`#!python return n + 2`</ParsonsLine>
<ParsonsLine>`#!python return 2`</ParsonsLine>
<ParsonsLine distractor>`#!python print("distractor")`</ParsonsLine>

</Parsons>

Parsons problems can also be used for non code tasks.

<Parsons slug="parsons-no-code">
Place the lines in order on the right. Distractors must not be included.

<ParsonsLine>First line</ParsonsLine>
<ParsonsLine>Second line</ParsonsLine>
<ParsonsLine>Third line</ParsonsLine>
<ParsonsLine>Fourth line</ParsonsLine>
<ParsonsLine distractor>This is a distractor</ParsonsLine>
<ParsonsLine distractor>This is also a distractor</ParsonsLine>

You can also add text after the blocks.

</Parsons>

Another option is to provide only the required lines in a single column.

<Parsons slug="parsons-no-code-single-col" singleColumn>

In this type of question, the user doesn't have to worry about selecting the correct lines.

<ParsonsLine>First line</ParsonsLine>
<ParsonsLine>Second line</ParsonsLine>
<ParsonsLine>Third line</ParsonsLine>
<ParsonsLine>Fourth line</ParsonsLine>

<Answer>
  Answer components will only appear once the student submitted their answer.
</Answer>

</Parsons>

---
title: Syntax highlighting
---

You can wrap inline code between backticks (`` ` ``) to apply inline syntax highlighting. Example: `` `#!python print('Hello world')` `` will be rendered as `#!python print('Hello world')` (note that you have to start the code with `#!text #!LANGUAGE` for the highlighter to work).

You can also use [Fenced Code Blocks](https://docs.astro.build/en/guides/markdown-content/#syntax-highlighting), i.e., wrap multiline code between triple backticks (` ``` `). Example:

def func(arg1, arg2):
    """
    Docstrings
    """

    return m

    print(func(21, 32))
